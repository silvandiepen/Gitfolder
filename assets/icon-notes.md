# Icon Notes

## Direction

GitFolder should feel like a small macOS utility, not a heavy developer product.

Possible icon metaphors:

- Folder with subtle branch lines.
- Folder with snapshot layers.
- Folder with circular sync arrow.
- Folder with commit dots.
- Folded folder corner with a Git node graph.

## Avoid

- Too much GitHub branding.
- Octocat references.
- Terminal-heavy visuals.
- Cloud backup clichés.

## Visual tone

- Simple.
- Mac-like.
- Utility-first.
- Clear at status bar size.
